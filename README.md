## 📌 Contexto

Este projeto representa uma empresa fictícia. A **Raízes do Nordeste** é uma rede de lanchonetes especializada em culinária nordestina, em expansão por diferentes capitais e cidades do interior do Brasil.

Com o crescimento da rede, surgiu a necessidade de uma solução multicanal (app, web e totem de autoatendimento) capaz de oferecer uma experiência consistente ao cliente, contemplando consulta ao cardápio, realização e acompanhamento de pedidos, programa de fidelidade e conformidade com a LGPD no tratamento de dados pessoais.

> Projeto desenvolvido como parte do Projeto Multidisciplinar da Trilha Front-End, no curso de Análise e Desenvolvimento de Sistemas da UNINTER (2026/1).
